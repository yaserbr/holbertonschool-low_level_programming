# holbertonschool-low_level_programming

This repository contains projects and exercises completed as part of the Holberton School Low-Level Programming curriculum.

## Topics Covered

* C Programming
* GCC Compilation
* Shell Scripts
* Betty Coding Style
* Functions
* Variables
* Pointers
* Arrays
* Memory Management
* File I/O
* Data Structures

Each project is organized in its own directory and focuses on specific low-level programming concepts.
